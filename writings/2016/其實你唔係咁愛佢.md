# 其實你唔係咁愛佢

```
如果你鍾意一個人，
鍾意到好辛苦；
其實你唔係愛佢，
你只不過係渴望擁有，
但又唔甘心
付出所有一切；
當你願意付出所有，
並接受佢嘅一切，
你內心唔會爭扎，
冇矛盾、冇惡念，
亦唔會想改變佢嘅意志；
默默地等，
就算天荒地老，
似乎好傻，
但真愛就係
唔計較利害；
愛就係選擇，
選擇就係捨棄 —
如果連未來嘅
其他可能性
都唔肯捨棄，
咁所謂愛情，
只不過係一份
用感情作交易嘅
合約合同；
呢份合同，
冇法律效力，
如果一定要
天長地久，
你只好用
各種手段
嘩鬧、
協迫、
誘惑、
欺騙、
自殘，
然後你嘅心就
陷入無盡黑暗之中；
如果你願意，
就聽由天命；
如果唔願意，
就唔好勉強自己，
迫出一個和諧景象。
```

（原文於 2016 年 4 月發佈，略有修輯）
